## Disclosure
For UI parts of this project, especially the parts which needed deep understanding of the Qt library, its different classes and methods, I used the help of AI tools and ressources.
Examples where I used AI tools:
- creating the /ui/material.qss file to create a similar UI style like Android Material You
- evaluating and double check my design/architectural decisions I made for the InfoPanel, CustomFilterProxModel or in the Mainw classes

On the other hand I didnt used Qt Creator to create the UI elements or to make any connections between UI elements.
I took a bit of use of Qt Designer to create dummy Layouts, to get a feeling how I can design the UI.

Furthermore I want to make clear, that I did not use any AI tools for the core implementation of the underlying data model of the project. This includes files in the /core and /manager directories.
